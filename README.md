# Шаблон сборки Webpack 5

Сборка проекта с помощью webpack и библиотеки пакетов NPM.

###В ней есть оптимизация и минимизация:

1. JavaScript (babel, core-js)
2. SCSS
3. PostCss
4. HTML
5. Webpack-dev-server
----
###Установка

- Клонируем к себе репозиторий
- Сделайте git clone 
- Смените директорию cd webpack-boilerplate
- Запустите npm install

----
###Для разработки:
- даем команду npm run dev
- Получаем комфортную среду для отладки
